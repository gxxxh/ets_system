### 调度比较界面
#### 接口定义
```
# Get方法
ip:port/perf/schedule/compare
```

#### 无参数

#### 期待展示效果
上面是表格（确定只有三行），下面使用柱状图展示，都是确定的数据
![](./img/schedule_compare.png)

#### 返回结果实例
1. JCT表格里写“任务总体完成时间（JCT）”
2. makespan表格里写“任务最大完成时间（makespan）”
3. predict翻译为“基于预测迭代时间（ours）”
4. measure翻译为“基于实际迭代时间(measure)”
5. random翻译为“基于随机迭代时间(random)”
```
{
    "message": "success",
    "data": {
        "predict": {
            "JCT": 27553.191616,
            "makespan": 2155.33526
        },
        "measure": {
            "JCT": 26190.47093267376,
            "makespan": 2076.064280042317
        },
        "random": {
            "JCT": 31707.78228163878,
            "makespan": 2462.656916608953
        }
    }
}
```
